# SMEAppHouse.Core.PuppeteerAdapter

## Overview

`SMEAppHouse.Core.PuppeteerAdapter` is a library for handling Puppeteer functionalities in .NET 8.0 applications. It provides adapters and utilities for working with headless browser automation.

**Target Framework**: .NET 8.0  
**Namespace**: `SMEAppHouse.Core.PuppeteerAdapter`

- PuppeteerSharp (v18.0.1)

- System.Net.NameResolution (v4.3.0)
- SMEAppHouse.Core.CodeKits
