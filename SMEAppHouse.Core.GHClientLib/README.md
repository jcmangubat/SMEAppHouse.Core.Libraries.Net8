# SMEAppHouse.Core.GHClientLib

## Overview

`SMEAppHouse.Core.GHClientLib` is a GraphHopper Extension Library for .NET 8.0 applications. It provides client functionality for interacting with GraphHopper routing services.

**Target Framework**: .NET 8.0  
**Namespace**: `SMEAppHouse.Core.GHClientLib`
