# SMEAppHouse.Core.FreeIPProxy

## Overview

`SMEAppHouse.Core.FreeIPProxy` is a library for generating usable proxy IP addresses for making HTTP requests anonymously in .NET 8.0 applications.

**Target Framework**: .NET 8.0  
**Namespace**: `SMEAppHouse.Core.FreeIPProxy`
