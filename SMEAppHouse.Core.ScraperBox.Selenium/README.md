# SMEAppHouse.Core.ScraperBox.Selenium

## Overview

`SMEAppHouse.Core.ScraperBox.Selenium` is a library for handling Selenium functionalities in .NET 8.0 applications. It provides utilities and extensions for web scraping and browser automation using Selenium WebDriver.

**Target Framework**: .NET 8.0  
**Namespace**: `SMEAppHouse.Core.ScraperBox.Selenium`

---

## Dependencies

- Selenium.WebDriver (v4.22.0)
- Selenium.WebDriver.ChromeDriver (v131.0.6778.8700)
- SMEAppHouse.Core.ScraperBox
